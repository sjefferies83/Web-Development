Web Design Cheat Sheet

Webpack Commands

npm install webpack -g

Installs webpack globally

npm install webpack --save

Installs webpack in project

webpack ./app.js app.bundle.js

Compiles app.js into app.bundle.js

Webpack Structure

Original source files need to be stored in a folder called src

Production compiled files need to be stored in a fodler called dist

A webpack.config.js files needs to be created to map to these files correctly

Babel Commands

npm install babel-core babel-preset-es2015 --save-dev

Installs babel and es6 support to project

npm install babel-loader --save-dev

Installs babel loader

npm install

installs modules

Babel Structure

A .babelrc file needs to be created

Modules need to be linked to app.js

Use Webpack to compile the app.js

Link to the compiled file in the html document
